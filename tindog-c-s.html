tindog

It's like tinder, but for dog's.

Project Background

Tindog is a made up A.K.A fake start up website, advertising a supposed IOS and Android application new on the market. The supposed application is based and built around the already pre-existing application known as "Tinder" a highly popular match-making platform intended to connect single's. The idea behind the tindog app is precisely the same as the exisiting tinder app in that it was designed to help "match" single and lonely dog's via the use of the dog owner's.

Challenges & Requirements

Tindog is a personal project created for learning purposes. The project was intended to test my coding abilities with HTML5 and CSS3, along with the current version of Bootstrap V5 (the current version of Bootstrap during the year of 2021). The challenges required are as follows:

  • Build a well structured and readable HTML website
  • Install and apply the Bootstrap framework
  • Make the website responsive.(well designed layout for all devices)
  • Refactor website styling via the D.R.Y method
